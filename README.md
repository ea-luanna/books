# Indicações de Livros para Programação

Este repositório apresenta uma seleção pessoal de livros voltados à programação, resultado da leitura de diversas obras e da avaliação de recomendações recebidas durante meu período acadêmico (com base na minha opnião pessoal). 

A lista prioriza obras que oferecem uma abordagem prática e compreensível dos fundamentos da programação. Os critérios de seleção incluem:

-  Clareza na exposição dos conceitos
-  Linguagem acessível para iniciantes
-  Didática voltada ao aprendizado progressivo

O conteúdo é atualizado continuamente, conforme novas leituras são realizadas e novos títulos relevantes são identificados.

## Algoritmos - Teoria e Prática
### por Thomas Thomas Cormen (Autor) 
<img src="https://github.com/user-attachments/assets/6f3ee136-7452-4af1-bfd3-65b56cdaf02c" width="250">



Referência clássica em algoritmos, este livro oferece uma abordagem completa e profunda sobre o tema. Embora tenha um estilo mais acadêmico, ele cobre praticamente tudo o que um estudante precisa saber: análise de complexidade, estruturas de dados, algoritmos clássicos e técnicas avançadas. É ideal para quem deseja se aprofundar e construir uma base sólida para estudos mais avançados.

## JavaScript: O Guia Definitivo 
### por David Flanagan (Autor), João Eduardo Nóbrega Tortello (Tradutor), Luciana Nedel
<img src="https://github.com/user-attachments/assets/1e2beeeb-1b01-4cfb-9bf9-3ea165a4eb1c" width="500">

Indico esse livro para servir como um guia nos estudos, ele engloba todos os conteúdos de forma completa. É bom para estudos com outras fontes, pois o livro possui uma linguagem mais técina, contudo, tem uma orientação boa, funciona realmente como um guia definitivo sobre javascript. Apesar da linguagem mais densa, ele é extremamente útil como material de consulta e aprofundamento, especialmente para quem já tem alguma experiência e quer entender o JavaScript desde o básico ao avançado.
