# Indicações de Livros para Programação

Este repositório apresenta uma seleção pessoal de livros voltados à programação, resultado da leitura de diversas obras e da avaliação de recomendações recebidas durante meu período acadêmico.

A lista prioriza obras que oferecem uma abordagem prática e compreensível dos fundamentos da programação. Os critérios de seleção incluem:

-  Clareza na exposição dos conceitos
-  Linguagem acessível para iniciantes
-  Didática voltada ao aprendizado progressivo

O conteúdo é atualizado continuamente, conforme novas leituras são realizadas e novos títulos relevantes são identificados.
